Feedstock for [cuba](https://feynarts.de/cuba) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Cuba – a library for multidimensional numerical integration

## Install

```bash
conda install -c hep-forge -c conda-forge cuba
```
